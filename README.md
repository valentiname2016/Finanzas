# Finanzas

Este proyecto consiste en el desarrollo de un simulador de portafolio de inversiones implementado en Python, cuyo objetivo es permitir a los usuarios analizar y experimentar con diferentes alternativas de inversión sin asumir riesgos reales.

El sistema utiliza la librería yfinance para obtener datos reales del mercado financiero, como precios históricos de acciones, lo que permite realizar simulaciones más cercanas a la realidad.

A través de este simulador, el usuario puede consultar acciones disponibles, simular la compra de acciones, realizar inversiones en CDT, evaluar el comportamiento de su portafolio y analizar la rentabilidad obtenida.

El sistema cuenta con un menú que permite navegar entre las distintas funcionalidades:

1. Ver acciones disponibles
2. Agregar acciones
3. Comprar acciones
4. Invertir en CDT
5. Realizar simulación
6. Ver resumen de rentabilidad
7. Visualizar gráfica de resultados
8. Salir

Finalmente, el simulador permite al usuario tomar decisiones informadas al comparar diferentes escenarios de inversión, apoyándose en visualizaciones gráficas y métricas de rentabilidad.
